# AegisForge Research

This directory is the project's research notebook. It captures novel ideas, open questions,
evaluation designs, and publication opportunities as they arise, so nothing is lost between
milestones.

Each note is a short, self-contained document. Notes may start as rough hypotheses and mature
into experiment designs and, eventually, paper sections.

## How to add a note

1. Create `NNN-short-slug.md` with a clear problem statement, hypothesis, proposed approach,
   evaluation idea, and open questions.
2. Add it to the index above.
3. Link related notes and the code that implements or tests the idea.
