# cortexward-eval

The benchmark-first evaluation harness for
[CortexWard](https://github.com/amarjaleelbanbhan/CortexWard): the `RunManifest` provenance
record, the deterministic finding-to-ground-truth matcher, and detection-quality metrics
(precision/recall/F1/FPR/FNR). Built before the heavy agent work (Phase 4) so every subsequent
capability is measured from the moment it exists (ADR-0007).

See [docs/benchmark/evaluation-framework.md](../../docs/benchmark/evaluation-framework.md) for
the full specification.
