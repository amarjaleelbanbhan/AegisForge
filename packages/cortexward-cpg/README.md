# cortexward-cpg

The Code Property Graph engine for [CortexWard](https://github.com/amarjaleelbanbhan/CortexWard):
a language-agnostic graph implementation of the `CodeGraph` port (AST + control-flow + data-flow
+ call graph), plus the Python reference `LanguageProvider`.

See [MPS §12](../../docs/specifications/MPS-v1.0.md#12-code-property-graph-specification) for the
full specification.
